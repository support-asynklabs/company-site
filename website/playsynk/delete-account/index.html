PlaySynk Account Deletion Request

How to request deletion of your PlaySynk account and personal data

Users can request deletion of their PlaySynk account and associated personal data by emailing support@asynklabs.com.

Please include

  • Registered email address
  • User name (if available)

Upon verification, the account and associated personal data will be deleted within 7 business days.

Data deleted

  • User profile
  • Account information
  • Game history
  • Reward information

Some data may be retained if required by law or for fraud prevention purposes.
